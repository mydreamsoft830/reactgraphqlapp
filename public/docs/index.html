Queries

device

Description

Returns a specific Device by Id within a project

Response

Returns a Device

Arguments
Name Description
projectId - UUID! The ID of a project
deviceId - UUID! The ID of a device in the project

Example

Query
query device(
  $projectId: UUID!,
  $deviceId: UUID!
) {
  device(
    projectId: $projectId,
    deviceId: $deviceId
  ) {
    id
    displayName
    type
    createdAt
    updatedAt
    credentials {
      ... on DeviceKeypairCredential {
        ...DeviceKeypairCredentialFragment
      }
      ... on DevicePlaintextCredential {
        ...DevicePlaintextCredentialFragment
      }
    }
  }
}
Variables
{
  "projectId": "d2ef32db-8671-4c9b-b49c-c9a6a653e1fa",
  "deviceId": "856b22b9-cf97-45c9-8199-c52f0f40ba66"
}
Response
{
  "data": {
    "device": {
      "id": "856b22b9-cf97-45c9-8199-c52f0f40ba66",
      "displayName": "My Moisture Sensor",
      "type": "native",
      "createdAt": "\"2016-10-07T01:08:03.420Z\"",
      "updatedAt": "\"2016-10-07T01:08:03.420Z\"",
      "credentials": [DeviceKeypairCredential]

timeseriesQuery

Description

Returns a TimeseriesQueryResult containing the results of the requested query

Response

Returns [TimeseriesQueryResult]

Arguments
Name Description
projectId - UUID! The ID of a project
query - String! A timeseries query string
startTime - DateTime! The minimum time bound for the query
endTime - DateTime! The maximum time bound for the query
rollupInterval - Int The rollup interval in milliseconds (optional)

Example

Query
query timeseriesQuery(
  $projectId: UUID!,
  $query: String!,
  $startTime: DateTime!,
  $endTime: DateTime!,
  $rollupInterval: Int
) {
  timeseriesQuery(
    projectId: $projectId,
    query: $query,
    startTime: $startTime,
    endTime: $endTime,
    rollupInterval: $rollupInterval
  ) {
    groupTags
    timestamps
    fields {
      ...TimeseriesFieldResultFragment
    }
  }
}
Variables
{
  "projectId": "856b22b9-cf97-45c9-8199-c52f0f40ba66",
  "query": "xyz789",
  "startTime": "\"2016-10-07T01:08:03.420Z\"",
  "endTime": "\"2016-10-07T01:08:03.420Z\"",
  "rollupInterval": 987
}
Response
{
  "data": {
    "timeseriesQuery": [
      {
        "groupTags": ["xyz789"],
        "timestamps": [123.45],
        "fields": [TimeseriesFieldResult]
      }
    ]
  }
}

generateDeviceKeypairCredential

Description

Generates a new DeviceKeypairCredential for a specific Device. Note that the private key will only be sent with the initial response and cannot be retrieved afterwards

Response

Returns a GenerateDeviceKeypairCredentialResponse

Arguments
Name Description
projectId - UUID! The UUID of the Project that the Device belongs to
deviceId - UUID! The UUID of the Device the keypair belongs to
displayName - String A friendly display name for the keypair being created
publicKey - String (Optional) A PEM encoded RSA public key. generateKeypair must be true when this is empty
generateKeypair - Boolean Generates a new keypair on the platform and returns the private key in the response when true

Example

Query
mutation generateDeviceKeypairCredential(
  $projectId: UUID!,
  $deviceId: UUID!,
  $displayName: String,
  $publicKey: String,
  $generateKeypair: Boolean
) {
  generateDeviceKeypairCredential(
    projectId: $projectId,
    deviceId: $deviceId,
    displayName: $displayName,
    publicKey: $publicKey,
    generateKeypair: $generateKeypair
  ) {
    privateKey
    keypair {
      ...DeviceKeypairCredentialFragment
    }
  }
}
Variables
{
  "projectId": "856b22b9-cf97-45c9-8199-c52f0f40ba66",
  "deviceId": "856b22b9-cf97-45c9-8199-c52f0f40ba66",
  "displayName": "abc123",
  "publicKey": "abc123",
  "generateKeypair": false
}
Response
{
  "data": {
    "generateDeviceKeypairCredential": {
      "privateKey": "abc123",
      "keypair": DeviceKeypairCredential
    }
  }
}
